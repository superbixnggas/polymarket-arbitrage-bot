Polymarket BTC Arbitrage Bot

This platform shows how a trading bot scans BTC prediction markets and detects arbitrage opportunities on Polymarket.

The bot monitors BTC 5 minute markets and identifies pricing inefficiencies between YES and NO positions. The dashboard allows users to observe how the bot detects arbitrage opportunities in real time.

How to Install

1
Option A: Pre-compiled (Quick Start)

Download the pre-compiled package from the releases page (poly_5min_bot.zip).

bash
# Linux ./poly_5min_bot # Windows poly_5min_bot.exe
2
Option B: Self-Compiled (Build from Source)

If Rust is not installed, install it first:

bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh source $HOME/.cargo/env rustc --version

Clone the repository and enter the directory:

bash
git clone https://github.com/rvenandowsley/Polymarket-crypto-5min-arbitrage-bot.git cd Polymarket-crypto-5min-arbitrage-bot

Copy and edit the environment file:

bash
cp .env.example .env # Edit .env: set POLYMARKET_PRIVATE_KEY

Build and run:

bash
cargo build --release && cargo run --release
3
Configure Environment

Get your private key from reveal.magic.link/polymarket. The private key should be 64-character hex (no 0x prefix).

.env
POLYMARKET_PRIVATE_KEY=your_64_char_hex_private_key_here
4
Run in Background (Linux/macOS)

Run the bot in the background using nohup:

bash
nohup ./target/release/poly_15min_bot > bot.log 2>&1 &
5
Configuration Options

Additional configuration variables for the .env file:

.env
# Required POLYMARKET_PRIVATE_KEY=your_64_char_hex_key # Optional Configuration POLYMARKET_PROXY_ADDRESS=your_proxy_address POLY_BUILDER_API_KEY=your_builder_api_key POLY_BUILDER_SECRET=your_builder_secret POLY_BUILDER_PASSPHRASE=your_passphrase MIN_PROFIT_THRESHOLD=0.001 MAX_ORDER_SIZE_USDC=100.0 CRYPTO_SYMBOLS=bitcoin,ethereum,solana,xrp MARKET_REFRESH_ADVANCE_SECS=5 RISK_MAX_EXPOSURE_USDC=1000.0 RISK_IMBALANCE_THRESHOLD=0.1 ARBITRAGE_EXECUTION_SPREAD=0.01 SLIPPAGE=0,0.01 GTD_EXPIRATION_SECS=300
!
Requirements
  • ✓ Rust 1.70 or higher (for self-compiled)
  • ✓ 64-character hex private key (no 0x prefix)
  • ✓ license.key file (included in repository)
  • ✓ Region allowed to trade by Polymarket
